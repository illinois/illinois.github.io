Illinois Open Source Software on github.com

github.com/illinois is a github organization for projects developed by students, staff, and faculty at The University of Illinois. Any students, staff, or faculty may create repositories on github.com/illinois — e-mail waf@illinois.edu using your @illinois.edu account and provide your github username to be added to the organization.

There are only a few guidelines we maintain within this organization:

  • Projects must be original, open source work. (Use your own personal github for forks, open course projects, etc)
  • Projects must contain a README file. (Don't dump random code in @illinois.)
  • This work should be something you are proud to be associated with you and Illinois. If you're not sure, it is best to host it on your own github account.

That's it! We encourage the use of the University of Illinois/NCSA Open Source License if those terms fit your needs, but this is not required.

Projects hosted on github.com/illinois are not endorsed by The University of Illinois in any way.

Happy coding!
