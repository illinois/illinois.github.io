Illinois Open Source Software on github.com

github.com/illinois was used a GitHub organization for projects developed by students, staff, and faculty at The University of Illinois. If you're at Illinois today, visit https://web.uillinois.edu/github for information about the University of Illinois System Shared GitHub Service.

Most active projects on this GitHub are from Wade Fagen-Ulmschneider's lab, which maintains this GitHub organization.

Projects hosted on github.com/illinois are not endorsed by The University of Illinois in any way.
